## 使用方法
右上方Code下拉栏选择Download ZIP 并 解压缩。  

Chrome浏览器地址栏输入chrome://extensions/         

打开开发者模式。

点击“加载已解压的扩展程序”按钮。    

浏览并选择刚才解压出来的文件夹。 

一旦选择了正确的文件夹，Chrome将会自动加载并启用这个扩展程序。
